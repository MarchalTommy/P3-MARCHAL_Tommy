# P3 MARCHAL_Tommy
 Repository de rendu du projet 3
================================

Rendu de l'app Entrevoisins du projet 3 du parcours Développeur d'Applications Android.

Entrevoisins est une application de service entre voisins. Sur ce projet, nous ne travaillons que sur la gestion des listes de voisins

Cette application à été développée dans le but d'être compatible avec l'API 21 d'Android, avec les contraintes que cela implique.



**Fonctions existante :** -gestion d'une liste ( Ajout et suppression d'entrées)

**Fonctions ajoutées :** -gestion des favoris ( Ajout et retrait d'entrées existante aux favoris, visualisation par liste)
                     -page de détail



**Pour compiler :**

1/  Télécharger le repository : Cliquer sur "Code" 
                                    -Soit cloner le git grâce à la première option
                                    -Soit télécharger le ZIP
2/  Ouvrir le dossier récupéré avec Android Studio (application faite depuis Android Studio 4.0.1)
3/  Compiler (Onglet Run, cliquer sur Run 'app')



**Librairies utilisées :** EventBus 3.1.1, ButterKnife 9.0.0, Glide 4.9.0.

Git EventBus : https://github.com/greenrobot/EventBus

Git ButterKnife : https://github.com/JakeWharton/butterknife    **/!\** Obsolète, priviliégier View Binding d'Android Jetpack **/!\**

Git Glide : https://github.com/bumptech/glide
